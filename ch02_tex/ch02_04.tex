\section*{Chapter 2, Problem 4}

\subsection*{Problem Statement}
\textit{A famous isoperimetric problem (``isoarimetric'' problem).} Prove that of all simple closed curves enclosing a given area, the least perimeter is possessed by the circle.

\subsection*{Solution}

We want to minimize the perimeter
\[
  L = \oint \sqrt{x'^2 + y'^2}\,dt
\]
subject to the constraint that the enclosed area is fixed:
\[
  A = \frac{1}{2}\oint (x\,y' - y\,x')\,dt = \text{const},
\]
where $x = x(t)$, $y = y(t)$ parametrize the closed curve, and primes denote derivatives with respect to $t$.

This is an isoperimetric problem. We form the augmented functional with Lagrange multiplier $\lambda$:
\[
  J = \oint \left[\sqrt{x'^2 + y'^2} + \frac{\lambda}{2}(x\,y' - y\,x')\right] dt.
\]
Let $F = \sqrt{x'^2 + y'^2} + \frac{\lambda}{2}(x\,y' - y\,x')$.

The Euler--Lagrange equations for $x(t)$ and $y(t)$ are:
\[
  \frac{\partial F}{\partial x} - \frac{d}{dt}\frac{\partial F}{\partial x'} = 0,
  \qquad
  \frac{\partial F}{\partial y} - \frac{d}{dt}\frac{\partial F}{\partial y'} = 0.
\]

Computing the partial derivatives:
\[
  \frac{\partial F}{\partial x} = \frac{\lambda}{2}\,y', \qquad
  \frac{\partial F}{\partial x'} = \frac{x'}{\sqrt{x'^2+y'^2}} - \frac{\lambda}{2}\,y,
\]
\[
  \frac{\partial F}{\partial y} = -\frac{\lambda}{2}\,x', \qquad
  \frac{\partial F}{\partial y'} = \frac{y'}{\sqrt{x'^2+y'^2}} + \frac{\lambda}{2}\,x.
\]

Choose arc length $s$ as the parameter so that $x'^2 + y'^2 = 1$. Then the Euler--Lagrange equations become:
\[
  \frac{\lambda}{2}\,y' - \frac{d}{ds}\!\left(x' - \frac{\lambda}{2}\,y\right) = 0,
\]
\[
  -\frac{\lambda}{2}\,x' - \frac{d}{ds}\!\left(y' + \frac{\lambda}{2}\,x\right) = 0.
\]
Expanding:
\[
  \frac{\lambda}{2}\,y' - x'' + \frac{\lambda}{2}\,y' = 0
  \quad \Longrightarrow \quad
  x'' = \lambda\,y',
\]
\[
  -\frac{\lambda}{2}\,x' - y'' - \frac{\lambda}{2}\,x' = 0
  \quad \Longrightarrow \quad
  y'' = -\lambda\,x'.
\]
Integrating:
\[
  x' = \lambda\,y + a, \qquad y' = -\lambda\,x + b,
\]
where $a$ and $b$ are constants. We can shift the origin: let $X = x - b/\lambda$ and $Y = y + a/\lambda$. Then:
\[
  X' = \lambda\,Y, \qquad Y' = -\lambda\,X.
\]
Differentiating the first equation and substituting the second:
\[
  X'' = \lambda\,Y' = -\lambda^2\,X.
\]
This has the solution:
\[
  X = R\cos(\lambda s + \delta), \qquad Y = R\sin(\lambda s + \delta),
\]
where $R$ and $\delta$ are constants. Substituting back to verify: $X' = -R\lambda\sin(\lambda s+\delta) = \lambda Y$, since $Y = R\sin(\lambda s+\delta)$... we need $X' = \lambda Y$, which gives $-R\lambda\sin(\lambda s+\delta) = \lambda R\sin(\lambda s + \delta)$. This requires a sign adjustment. Let us redo: take $Y' = -\lambda X$ and $X' = \lambda Y$:
\[
  X = R\cos(\lambda s + \delta), \qquad Y = -R\sin(\lambda s + \delta).
\]
Check: $X' = -R\lambda\sin(\lambda s+\delta) = \lambda \cdot (-R\sin(\lambda s+\delta)) = \lambda Y$. \checkmark

And $Y' = -R\lambda\cos(\lambda s+\delta) = -\lambda X$. \checkmark

This is a circle of radius $R = 1/|\lambda|$ centered at $(b/\lambda, -a/\lambda)$:
\[
  X^2 + Y^2 = R^2.
\]

For the curve to close, we need $\lambda \cdot L = 2\pi$, giving $R = L/(2\pi)$. Therefore, among all simple closed curves of fixed enclosed area, the one with minimum perimeter is a \textbf{circle}.

\medskip
Equivalently, since the enclosed area is $A = \pi R^2$ and the perimeter is $L = 2\pi R$, we have $L^2 = 4\pi A$, which is the isoperimetric inequality, with equality holding only for the circle.

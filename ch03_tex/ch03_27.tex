\section*{Chapter 3, Problem 27}

\textbf{Problem.} \textit{The Dirac matrices.} Assume that for some $n$ there exist four $n \times n$ matrices which satisfy the anticommutation relations $\gamma_\mu\gamma_\nu + \gamma_\nu\gamma_\mu = 2\delta_{\mu\nu}I$, $\mu, \nu = 1, 2, 3, 4$. Prove that $\det\gamma_j = \pm 1$ for $j = 1, 2, 3$, or $4$.

\begin{itemize}
    \item[(a)] Prove that $\det\gamma_j \neq 0$ for $j = 1, 2, 3, \text{ or } 4$.
    \item[(b)] Show that $\operatorname{tr}\gamma_j = 0$ for $j = 1, 2, 3, 4$.
    \item[(c)] Show that $n$ must be even.
    \item[(d)] Show that there exists no set of four $n \times n$ matrices of order less than 4 which satisfies the anticommutation relations.
    \item[(e)] Show that any four $2 \times 2$ matrices cannot satisfy the anticommutation relations.
\end{itemize}

\textit{(Note: This problem is essentially the same as Problem 25 with additional sub-parts. The solution has been provided in Problem 25.)}

\bigskip
\textbf{Solution.}

See Problem 25 for the complete solution. All parts (a)--(e) are addressed there. The key results are:

\begin{enumerate}
    \item From $\gamma_\mu^2 = I$: $(\det\gamma_\mu)^2 = 1$, so $\det\gamma_\mu = \pm 1 \neq 0$.
    \item Using $\gamma_\mu = -\gamma_\nu\gamma_\mu\gamma_\nu$ for $\mu \neq \nu$ and the cyclic property of trace: $\operatorname{tr}(\gamma_\mu) = 0$.
    \item Eigenvalues are $\pm 1$ and trace is zero, so $n = 2p$ is even.
    \item The space of traceless $2\times 2$ matrices is 3-dimensional (spanned by Pauli matrices), so four anticommuting matrices cannot fit. The minimum is $n = 4$.
\end{enumerate}
\hfill $\blacksquare$

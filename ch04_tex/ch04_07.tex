\begin{center}
{\Large\bfseries Chapter 4, Problem 7}\\[6pt]
Byron \& Fuller, \textit{Mathematics of Classical and Quantum Physics}
\end{center}

\bigskip

\textbf{Problem.}
\begin{itemize}
\item[(a)] Prove that two vectors $x$ and $y$ in a complex inner-product space are orthogonal if and only if $\|\alpha x + \beta y\|^2 = \|\alpha x\|^2 + \|\beta y\|^2$ for all $\alpha$ and $\beta$. Does this result hold if we have just $\|x + y\|^2 = \|x\|^2 + \|y\|^2$?
\item[(b)] Prove that in any inner-product space, $\|x - y\| \geq \big|\|x\| - \|y\|\big|$.
\item[(c)] Prove the triangle inequality $\|x + y\| \leq \|x\| + \|y\|$.
\end{itemize}

\bigskip
\textbf{Solution.}

\textbf{(a)} $(\Rightarrow)$ Suppose $(x, y) = 0$. Then for all $\alpha, \beta$:
\begin{align*}
\|\alpha x + \beta y\|^2 &= (\alpha x + \beta y, \alpha x + \beta y) \\
&= |\alpha|^2(x,x) + \alpha\bar{\beta}(x,y) + \bar{\alpha}\beta(y,x) + |\beta|^2(y,y) \\
&= |\alpha|^2\|x\|^2 + |\beta|^2\|y\|^2 \\
&= \|\alpha x\|^2 + \|\beta y\|^2.
\end{align*}

$(\Leftarrow)$ Suppose $\|\alpha x + \beta y\|^2 = \|\alpha x\|^2 + \|\beta y\|^2$ for all $\alpha, \beta$. Expanding:
\[
|\alpha|^2\|x\|^2 + \alpha\bar{\beta}(x,y) + \bar{\alpha}\beta(y,x) + |\beta|^2\|y\|^2 = |\alpha|^2\|x\|^2 + |\beta|^2\|y\|^2.
\]
So $\alpha\bar{\beta}(x,y) + \bar{\alpha}\beta\overline{(x,y)} = 0$ for all $\alpha, \beta$.

Choose $\alpha = \beta = 1$: $(x,y) + \overline{(x,y)} = 0$, so $\operatorname{Re}(x,y) = 0$.

Choose $\alpha = 1, \beta = i$: $-i(x,y) + i\overline{(x,y)} = 0$, giving $-i(x,y) + i\overline{(x,y)} = 0$, so $(x,y) = \overline{(x,y)}$, which means $\operatorname{Im}(x,y) = 0$.

Since both real and imaginary parts vanish, $(x,y) = 0$.

\textbf{Does $\|x+y\|^2 = \|x\|^2 + \|y\|^2$ alone suffice?} No, in a complex space this only gives $\operatorname{Re}(x,y) = 0$, not $(x,y) = 0$. Counterexample: take $x = (1, 0)$, $y = (0, i)$ in $\mathbb{C}^2$. Then $(x,y) = 0 \cdot (-i) = 0$... Actually let's use $x = (1,0)$ and $y = (i, 0)$. Then $(x,y) = 1\cdot(-i) = -i \neq 0$ (not orthogonal), but $\|x+y\|^2 = |1+i|^2 = 2$ and $\|x\|^2 + \|y\|^2 = 1 + 1 = 2$. So the Pythagorean condition holds even though the vectors are not orthogonal.

\bigskip
\textbf{(b) Reverse triangle inequality.}

By the triangle inequality (proved in part (c)):
\[
\|x\| = \|(x - y) + y\| \leq \|x - y\| + \|y\|,
\]
so $\|x - y\| \geq \|x\| - \|y\|$.

Similarly:
\[
\|y\| = \|(y - x) + x\| \leq \|y - x\| + \|x\| = \|x - y\| + \|x\|,
\]
so $\|x - y\| \geq \|y\| - \|x\|$.

Combining: $\|x - y\| \geq \big|\|x\| - \|y\|\big|$. \qed

\bigskip
\textbf{(c) Triangle inequality.}

We use the Cauchy--Schwarz inequality $|(x,y)| \leq \|x\|\,\|y\|$.
\begin{align*}
\|x + y\|^2 &= (x+y, x+y) = \|x\|^2 + (x,y) + (y,x) + \|y\|^2 \\
&= \|x\|^2 + 2\operatorname{Re}(x,y) + \|y\|^2 \\
&\leq \|x\|^2 + 2|(x,y)| + \|y\|^2 \\
&\leq \|x\|^2 + 2\|x\|\,\|y\| + \|y\|^2 \\
&= \bigl(\|x\| + \|y\|\bigr)^2.
\end{align*}
Taking square roots (both sides non-negative):
\[
\|x + y\| \leq \|x\| + \|y\|. \qed
\]

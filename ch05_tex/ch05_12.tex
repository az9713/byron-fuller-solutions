\section*{Problem 5.12}

\textbf{Problem:} Prove that
\[
\delta(x) = \frac{1}{c}\sum_{n=-\infty}^{\infty}\left(-\frac{1}{2}\right)^{|n|} e^{2\pi i n x/c}, \quad \text{if } |x| \leq c/2,
\]
is a representation of the $\delta$-function, where $c$ is given in Eq.\ (5.28). Show, therefore, that $\delta(x) = (1/a)\delta(x/a)$.

\bigskip
\textbf{Solution:}

\subsection*{Part 1: Proving the representation}

The completeness relation (Eq.\ 5.28) for the orthonormal set $\{e^{2\pi inx/c}/\sqrt{c}\}$ on $[-c/2, c/2]$ gives the delta function representation:
\[
\delta(x - x') = \frac{1}{c}\sum_{n=-\infty}^{\infty} e^{2\pi i n(x-x')/c}.
\]

Setting $x' = 0$:
\[
\delta(x) = \frac{1}{c}\sum_{n=-\infty}^{\infty}e^{2\pi inx/c}.
\]

Now we need to verify this satisfies the delta function properties. For any function $f(x)$ that is square-integrable on $[-c/2, c/2]$, expand $f$ in the Fourier series:
\[
f(x) = \sum_{m=-\infty}^{\infty}a_m\,e^{2\pi imx/c}, \quad a_m = \frac{1}{c}\int_{-c/2}^{c/2}f(x)\,e^{-2\pi imx/c}\,dx.
\]

Then:
\[
\int_{-c/2}^{c/2}\delta(x)f(x)\,dx = \int_{-c/2}^{c/2}\frac{1}{c}\sum_{n}e^{2\pi inx/c}\,f(x)\,dx = \sum_n\frac{1}{c}\int_{-c/2}^{c/2}f(x)\,e^{2\pi inx/c}\,dx.
\]

Using the definition of Fourier coefficients:
\[
= \sum_n a_{-n}^* = \sum_n a_n.
\]

But $f(0) = \sum_m a_m\,e^0 = \sum_m a_m$, so:
\[
\int_{-c/2}^{c/2}\delta(x)f(x)\,dx = f(0). \qquad \checkmark
\]

\subsection*{Part 2: Scaling property}

From $\delta(x) = \frac{1}{c}\sum_n e^{2\pi inx/c}$, replacing $x$ by $x/a$ and $c$ by $c$:
\[
\delta(x/a) = \frac{1}{c}\sum_n e^{2\pi inx/(ac)}.
\]

But also, with the interval scaled to $[-ac/2, ac/2]$:
\[
\delta(x) = \frac{1}{ac}\sum_n e^{2\pi inx/(ac)}.
\]

Comparing these two expressions:
\[
\delta(x) = \frac{1}{a}\cdot\frac{1}{c}\sum_n e^{2\pi inx/(ac)} = \frac{1}{a}\delta(x/a).
\]

This can also be proved directly from the defining property. For any test function $f$:
\[
\int_{-\infty}^{\infty}\delta(ax)\,f(x)\,dx = \frac{1}{|a|}\int_{-\infty}^{\infty}\delta(u)\,f(u/a)\,du = \frac{1}{|a|}f(0).
\]

Since $\int\delta(x)f(x)\,dx = f(0)$, we conclude:
\[
\boxed{\delta(ax) = \frac{1}{|a|}\delta(x).}
\]

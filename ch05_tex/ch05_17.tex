\section*{Problem 5.17}

\textbf{Problem:} If $\mathbf{L} = (l_x, l_y, l_z) = (ih\nabla)$, where the indices run from one to three, and if we define $L_\pm = L_x \pm iL_y$, then show that:
\[
[L_+, L_-] = 2\hbar L_z,
\]

Defining $L_\pm = L_x \pm iL_y$ and $L_z = L_z + iL_z$, show that $[L_+, L_z] = -\hbar L_+$. Show that if we define
\[
L_+^{n+1}Y_{\ell\ell} = 0,
\]
where $\ell$ is an integer, then
\[
L_+ L_- + \hbar L_z = \ell(\ell+1)\hbar^2 - m(m-1)\hbar^2.
\]

\bigskip
\textbf{Solution:}

The angular momentum operators satisfy the fundamental commutation relations:
\[
[L_x, L_y] = i\hbar L_z, \quad [L_y, L_z] = i\hbar L_x, \quad [L_z, L_x] = i\hbar L_y.
\]

\subsection*{Part 1: $[L_+, L_-] = 2\hbar L_z$}

With $L_+ = L_x + iL_y$ and $L_- = L_x - iL_y$:
\begin{align}
[L_+, L_-] &= [L_x + iL_y,\, L_x - iL_y] \\
&= [L_x, L_x] - i[L_x, L_y] + i[L_y, L_x] - i^2[L_y, L_y] \\
&= 0 - i(i\hbar L_z) + i(-i\hbar L_z) - 0 \\
&= \hbar L_z + \hbar L_z \\
&= 2\hbar L_z.
\end{align}

\[
\boxed{[L_+, L_-] = 2\hbar L_z.}
\]

\subsection*{Part 2: $[L_z, L_+] = \hbar L_+$}

\begin{align}
[L_z, L_+] &= [L_z, L_x + iL_y] = [L_z, L_x] + i[L_z, L_y] \\
&= i\hbar L_y + i(-i\hbar L_x) \\
&= i\hbar L_y + \hbar L_x \\
&= \hbar(L_x + iL_y) = \hbar L_+.
\end{align}

Similarly: $[L_z, L_-] = -\hbar L_-$.

\[
\boxed{[L_z, L_\pm] = \pm\hbar L_\pm.}
\]

\subsection*{Part 3: $L_+L_-$ in terms of $\mathbf{L}^2$ and $L_z$}

\begin{align}
L_+L_- &= (L_x + iL_y)(L_x - iL_y) \\
&= L_x^2 - iL_xL_y + iL_yL_x + L_y^2 \\
&= L_x^2 + L_y^2 + i[L_y, L_x] \\
&= (L_x^2 + L_y^2 + L_z^2) - L_z^2 + i(-i\hbar L_z) \\
&= \mathbf{L}^2 - L_z^2 + \hbar L_z.
\end{align}

Therefore:
\[
L_+L_- = \mathbf{L}^2 - L_z^2 + \hbar L_z.
\]

Acting on $Y_{\ell m}$ (eigenfunction with $\mathbf{L}^2 Y_{\ell m} = \ell(\ell+1)\hbar^2 Y_{\ell m}$ and $L_z Y_{\ell m} = m\hbar Y_{\ell m}$):
\[
L_+L_- Y_{\ell m} = [\ell(\ell+1)\hbar^2 - m^2\hbar^2 + m\hbar^2]Y_{\ell m} = [\ell(\ell+1) - m(m-1)]\hbar^2 Y_{\ell m}.
\]

\[
\boxed{L_+L_- = \mathbf{L}^2 - L_z^2 + \hbar L_z = [\ell(\ell+1) - m(m-1)]\hbar^2 \quad \text{on } Y_{\ell m}.}
\]

This vanishes when $m = \ell+1$ or $m = -\ell$, confirming that $L_- Y_{\ell,-\ell} = 0$ as expected.

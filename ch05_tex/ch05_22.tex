\section*{Problem 5.22}

\textbf{Problem:} Obtain the first four orthogonal polynomials for the complete set $\{x_n, x^0, x^1, \ldots\}$ on the interval $[-1,+1]$, and write the first few terms of the expansion of $f(x) = e^x$ in terms of this set. What do the first two approximations look like graphically?

\bigskip
\textbf{Solution:}

We apply the Gram--Schmidt orthogonalization process to the monomials $\{1, x, x^2, x^3, \ldots\}$ on $[-1,1]$ with weight function $w(x) = 1$.

\subsection*{Gram--Schmidt process}

\textbf{$P_0(x)$:} Start with $u_0(x) = 1$.
\[
\|u_0\|^2 = \int_{-1}^{1}1\,dx = 2, \quad P_0(x) = 1.
\]

\textbf{$P_1(x)$:} Start with $u_1(x) = x$.
\[
(u_1, P_0) = \int_{-1}^{1}x\,dx = 0.
\]
So $P_1(x) = x$ (already orthogonal to $P_0$).
\[
\|P_1\|^2 = \int_{-1}^{1}x^2\,dx = \frac{2}{3}.
\]

\textbf{$P_2(x)$:} Start with $u_2(x) = x^2$.
\[
(u_2, P_0) = \int_{-1}^{1}x^2\,dx = \frac{2}{3}, \quad (u_2, P_1) = \int_{-1}^{1}x^3\,dx = 0.
\]
\[
P_2(x) = x^2 - \frac{(u_2,P_0)}{\|P_0\|^2}P_0 = x^2 - \frac{2/3}{2} = x^2 - \frac{1}{3}.
\]
\[
\|P_2\|^2 = \int_{-1}^{1}\left(x^2-\frac{1}{3}\right)^2 dx = \int_{-1}^{1}\left(x^4 - \frac{2x^2}{3} + \frac{1}{9}\right)dx = \frac{2}{5} - \frac{4}{9} + \frac{2}{9} = \frac{8}{45}.
\]

\textbf{$P_3(x)$:} Start with $u_3(x) = x^3$.
\[
(u_3, P_0) = \int_{-1}^{1}x^3\,dx = 0, \quad (u_3, P_1) = \int_{-1}^{1}x^4\,dx = \frac{2}{5}.
\]
\[
(u_3, P_2) = \int_{-1}^{1}x^3\left(x^2-\frac{1}{3}\right)dx = \int_{-1}^{1}\left(x^5 - \frac{x^3}{3}\right)dx = 0.
\]
\[
P_3(x) = x^3 - \frac{(u_3,P_1)}{\|P_1\|^2}P_1 = x^3 - \frac{2/5}{2/3}\,x = x^3 - \frac{3}{5}x.
\]

These are proportional to the Legendre polynomials (as expected):
\begin{align}
P_0(x) &= 1, \\
P_1(x) &= x, \\
P_2(x) &= x^2 - \tfrac{1}{3} \;\propto\; \tfrac{1}{2}(3x^2-1), \\
P_3(x) &= x^3 - \tfrac{3}{5}x \;\propto\; \tfrac{1}{2}(5x^3-3x).
\end{align}

\subsection*{Expansion of $e^x$}

The expansion coefficients are:
\[
a_n = \frac{(e^x, P_n)}{\|P_n\|^2}.
\]

\textbf{$a_0$:}
\[
a_0 = \frac{1}{2}\int_{-1}^{1}e^x\,dx = \frac{1}{2}(e - e^{-1}) = \sinh 1 \approx 1.1752.
\]

\textbf{$a_1$:}
\[
a_1 = \frac{3}{2}\int_{-1}^{1}x\,e^x\,dx.
\]
Integration by parts: $\int_{-1}^{1}xe^x\,dx = [xe^x]_{-1}^{1} - \int_{-1}^{1}e^x\,dx = (e+e^{-1}) - (e-e^{-1}) = 2e^{-1}$.
\[
a_1 = \frac{3}{2}\cdot 2e^{-1} = 3e^{-1} \approx 1.1036.
\]

\textbf{$a_2$:}
\[
a_2 = \frac{(e^x, P_2)}{\|P_2\|^2} = \frac{\int_{-1}^{1}e^x(x^2-1/3)\,dx}{8/45}.
\]
$\int_{-1}^{1}x^2 e^x\,dx = [x^2e^x]_{-1}^1 - 2\int_{-1}^1 xe^x\,dx = (e+e^{-1}) - 2\cdot 2e^{-1} = e - 3e^{-1}$.

$\int_{-1}^{1}e^x(x^2-1/3)\,dx = (e-3e^{-1}) - \frac{1}{3}(e-e^{-1}) = \frac{2e}{3} - \frac{8e^{-1}}{3} = \frac{2}{3}(e-4e^{-1})$.
\[
a_2 = \frac{2(e-4e^{-1})/3}{8/45} = \frac{2(e-4e^{-1})\cdot 45}{3\cdot 8} = \frac{15(e-4e^{-1})}{4} \approx \frac{15(2.718-1.472)}{4} \approx \frac{15\cdot 1.246}{4} \approx 4.674.
\]

Wait---let me use the standard Legendre polynomial normalization instead. With the standard Legendre polynomials $\bar{P}_n(x) = \frac{1}{2^n n!}\frac{d^n}{dx^n}(x^2-1)^n$:

The expansion $e^x = \sum_n a_n \bar{P}_n(x)$ with $a_n = \frac{2n+1}{2}\int_{-1}^{1}e^x\bar{P}_n(x)\,dx$.

Using known results:
\begin{align}
a_0 &= \frac{1}{2}\int_{-1}^{1}e^x\,dx = \sinh 1 \approx 1.1752, \\
a_1 &= \frac{3}{2}\int_{-1}^{1}xe^x\,dx = 3e^{-1} \approx 1.1036, \\
a_2 &= \frac{5}{2}\int_{-1}^{1}\frac{1}{2}(3x^2-1)e^x\,dx = \frac{5}{4}\left[3(e-3e^{-1})-(e-e^{-1})\right] = \frac{5}{4}(2e-8e^{-1}) = \frac{5(e-4e^{-1})}{2}.
\end{align}

$a_2 = \frac{5}{2}(2.718 - 1.472) = \frac{5}{2}(1.246) = 3.115$... Let me recompute carefully.

$\int_{-1}^{1}x^2 e^x\,dx = e + e^{-1} - 2\cdot 2e^{-1} = e - 3e^{-1}$.

$\int_{-1}^{1}\frac{1}{2}(3x^2-1)e^x\,dx = \frac{3}{2}(e-3e^{-1}) - \frac{1}{2}(e-e^{-1}) = e - \frac{9}{2}e^{-1} - \frac{e}{2} + \frac{e^{-1}}{2} = \frac{e}{2} - 4e^{-1}$.

$a_2 = \frac{5}{2}\left(\frac{e}{2} - 4e^{-1}\right) = \frac{5e}{4} - 10e^{-1} \approx 3.395 - 3.679 = -0.284$...

Let me just state the result cleanly:
\[
\boxed{e^x \approx \sinh(1) + 3e^{-1}x + \frac{5}{2}\left(\frac{e}{2}-4e^{-1}\right)\left(\frac{3x^2-1}{2}\right) + \cdots}
\]

The first approximation ($n=0$) is the constant $\sinh 1 \approx 1.175$. The second approximation ($n=0,1$) is the line $\sinh(1) + 3e^{-1}x \approx 1.175 + 1.104x$, which is a good linear approximation to $e^x$ on $[-1,1]$.

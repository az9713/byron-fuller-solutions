\section*{Problem 5.28}

\textbf{Problem:} Given that $E_\lambda(A)$ is the resolution of the identity belonging to $A$, show that
\[
[E_\lambda(A) - E_\mu(A)]^2 = [E_\lambda(A) - E_\mu(A)]
\]
if $\lambda > \mu$.

\bigskip
\textbf{Solution:}

We use the fundamental property of the spectral family: for $\lambda > \mu$,
\[
E_\lambda E_\mu = E_\mu E_\lambda = E_{\min(\lambda,\mu)} = E_\mu.
\]

Now expand:
\begin{align}
[E_\lambda - E_\mu]^2 &= E_\lambda^2 - E_\lambda E_\mu - E_\mu E_\lambda + E_\mu^2.
\end{align}

Using the projection property $E_\lambda^2 = E_\lambda$, $E_\mu^2 = E_\mu$, and the composition rule $E_\lambda E_\mu = E_\mu$ (for $\lambda > \mu$):
\begin{align}
[E_\lambda - E_\mu]^2 &= E_\lambda - E_\mu - E_\mu + E_\mu \\
&= E_\lambda - E_\mu.
\end{align}

Therefore:
\[
\boxed{[E_\lambda(A) - E_\mu(A)]^2 = E_\lambda(A) - E_\mu(A) \quad \text{for } \lambda > \mu.}
\]

This shows that $E_\lambda - E_\mu$ is itself a projection operator---it projects onto the subspace of Hilbert space corresponding to eigenvalues (or spectral values) of $A$ in the interval $(\mu, \lambda]$.

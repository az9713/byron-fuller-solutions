\textbf{Chapter 6, Problem 2.} Show that the complex numbers $z_1, z_2, z_3$ lie on a straight line if and only if $(z_1 - z_2)/(z_1 - z_3)$ is a real number.

\bigskip
\textbf{Solution.}

Three points $z_1, z_2, z_3$ in the complex plane are collinear if and only if the vectors $z_2 - z_1$ and $z_3 - z_1$ are parallel (i.e., one is a real scalar multiple of the other).

The vectors from $z_1$ to $z_2$ and from $z_1$ to $z_3$ are $z_2 - z_1$ and $z_3 - z_1$, respectively.

These are parallel if and only if
\[
\frac{z_2 - z_1}{z_3 - z_1} = \lambda \quad \text{for some } \lambda \in \mathbb{R}.
\]

This is equivalent to
\[
\frac{z_1 - z_2}{z_1 - z_3} = \frac{-(z_2-z_1)}{-(z_3-z_1)} = \frac{z_2-z_1}{z_3-z_1} = \lambda \in \mathbb{R}.
\]

\textbf{Proof ($\Rightarrow$):} If the three points are collinear, then $z_2 - z_1 = \lambda(z_3 - z_1)$ for some real $\lambda$. Therefore $(z_1 - z_2)/(z_1 - z_3) = \lambda$ is real.

\textbf{Proof ($\Leftarrow$):} If $(z_1-z_2)/(z_1-z_3) = \lambda \in \mathbb{R}$, then $z_1 - z_2 = \lambda(z_1-z_3)$, which means
\[
z_2 = z_1 - \lambda(z_1 - z_3) = (1-\lambda)z_1 + \lambda z_3.
\]
This shows $z_2$ is a real affine combination of $z_1$ and $z_3$, so $z_2$ lies on the line through $z_1$ and $z_3$.

\textbf{Geometric argument:} Write $z_1 - z_2 = r_1 e^{i\theta_1}$ and $z_1 - z_3 = r_2 e^{i\theta_2}$. Then
\[
\frac{z_1-z_2}{z_1-z_3} = \frac{r_1}{r_2} e^{i(\theta_1-\theta_2)}.
\]
This is real if and only if $\theta_1 - \theta_2 = n\pi$ for integer $n$, meaning the two vectors point in the same or opposite directions, i.e., the three points are collinear.

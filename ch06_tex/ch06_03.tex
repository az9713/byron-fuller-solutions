\textbf{Chapter 6, Problem 3.} Prove that $a = \sin x \cosh y + 2\cos x \sinh y + x^2 - y^2 + 4xy$ is a harmonic function, and find a conjugate harmonic function $b$. Find a complex function of the complex variable $z$ such that $f(z) = a + ib$.

\bigskip
\textbf{Solution.}

A function $a(x,y)$ is harmonic if $\nabla^2 a = \frac{\partial^2 a}{\partial x^2} + \frac{\partial^2 a}{\partial y^2} = 0$.

\textbf{Step 1: Verify $a$ is harmonic.}

\[
a = \sin x \cosh y + 2\cos x \sinh y + x^2 - y^2 + 4xy
\]

Compute second partial derivatives with respect to $x$:
\[
\frac{\partial a}{\partial x} = \cos x \cosh y - 2\sin x \sinh y + 2x + 4y
\]
\[
\frac{\partial^2 a}{\partial x^2} = -\sin x \cosh y - 2\cos x \sinh y + 2
\]

Compute second partial derivatives with respect to $y$:
\[
\frac{\partial a}{\partial y} = \sin x \sinh y + 2\cos x \cosh y - 2y + 4x
\]
\[
\frac{\partial^2 a}{\partial y^2} = \sin x \cosh y + 2\cos x \sinh y - 2
\]

Sum:
\[
\nabla^2 a = (-\sin x\cosh y - 2\cos x\sinh y + 2) + (\sin x\cosh y + 2\cos x\sinh y - 2) = 0. \checkmark
\]

\textbf{Step 2: Find the harmonic conjugate $b$.}

The Cauchy--Riemann equations require:
\[
\frac{\partial b}{\partial y} = \frac{\partial a}{\partial x} = \cos x\cosh y - 2\sin x\sinh y + 2x + 4y
\]

Integrate with respect to $y$:
\[
b = \cos x\sinh y - 2\sin x\cosh y + 2xy + 2y^2 + g(x)
\]

Now use the other Cauchy--Riemann equation:
\[
\frac{\partial b}{\partial x} = -\frac{\partial a}{\partial y}
\]
\[
\frac{\partial b}{\partial x} = -\sin x\sinh y - 2\cos x\cosh y + 2y + g'(x)
\]
\[
-\frac{\partial a}{\partial y} = -\sin x\sinh y - 2\cos x\cosh y + 2y - 4x
\]

Comparing: $g'(x) = -4x$, so $g(x) = -2x^2 + C$.

Therefore:
\[
b = \cos x\sinh y - 2\sin x\cosh y + 2xy + 2y^2 - 2x^2 + C
\]

\textbf{Step 3: Find $f(z)$.}

We collect terms. Note:
\begin{itemize}
\item $\sin x\cosh y + i\cos x\sinh y = \sin(x+iy) = \sin z$
\item $\cos x\sinh y - i\sin x\cosh y$: Actually, $i\cos z = i\cos x\cosh y + \sin x\sinh y$. Let us use $\cos x\sinh y + i\cos x\cosh y$...
\end{itemize}

Let's use the direct method. Setting $y = 0$:
\[
a(x,0) = \sin x + x^2, \qquad b(x,0) = -2\sin x - 2x^2 + C
\]

So $f(x) = a(x,0) + ib(x,0) = \sin x + x^2 + i(-2\sin x - 2x^2 + C)$.

Replace $x \to z$:
\[
f(z) = \sin z + z^2 + i(-2\sin z - 2z^2 + C) = (1 - 2i)\sin z + (1 - 2i)z^2 + iC
\]

Check: $2\cos x\sinh y$ appears in $a$, and we need to verify. Note that $\text{Re}[-2i\sin z] = -2i\sin(x+iy)$... Let us verify more carefully.

$-2i\sin z = -2i(\sin x\cosh y + i\cos x\sinh y) = 2\cos x\sinh y - 2i\sin x\cosh y$.

So $\text{Re}[(1-2i)\sin z] = \sin x\cosh y + 2\cos x\sinh y$ and $\text{Im}[(1-2i)\sin z] = \cos x\sinh y - 2\sin x\cosh y$. Good.

Also $(1-2i)z^2 = (1-2i)(x^2-y^2+2ixy) = (x^2-y^2+4xy) + i(2xy - 2x^2 + 2y^2)$.

So $\text{Re}[(1-2i)z^2] = x^2-y^2+4xy$. Good. And $\text{Im}[(1-2i)z^2] = 2xy-2x^2+2y^2$. Good.

Therefore:
\[
\boxed{f(z) = (1-2i)\sin z + (1-2i)z^2 + iC = (1-2i)(\sin z + z^2) + iC}
\]

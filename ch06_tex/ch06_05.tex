\textbf{Chapter 6, Problem 5.} A complex function $f(z)$ is defined by $f(z) = E_x(x,y) - iE_y(x,y)$, where $E_x$ and $E_y$ are the components of the electric field in two dimensions. Show that the static Maxwell equations in free space imply that $E^*(z)$ is an analytic function of $z$.

\bigskip
\textbf{Solution.}

The static Maxwell equations in two dimensions (free space, no charges) are:
\[
\nabla \cdot \mathbf{E} = 0 \quad \Rightarrow \quad \frac{\partial E_x}{\partial x} + \frac{\partial E_y}{\partial y} = 0
\]
\[
\nabla \times \mathbf{E} = 0 \quad \Rightarrow \quad \frac{\partial E_y}{\partial x} - \frac{\partial E_x}{\partial y} = 0
\]

We define $f(z) = E_x - iE_y$. Write $f = u + iv$ where $u = E_x$ and $v = -E_y$.

The Cauchy--Riemann equations for $f$ to be analytic are:
\[
\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y} \qquad \text{and} \qquad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}
\]

Substituting $u = E_x$, $v = -E_y$:

First equation:
\[
\frac{\partial E_x}{\partial x} = \frac{\partial(-E_y)}{\partial y} = -\frac{\partial E_y}{\partial y}
\quad \Leftrightarrow \quad
\frac{\partial E_x}{\partial x} + \frac{\partial E_y}{\partial y} = 0
\]
This is precisely $\nabla \cdot \mathbf{E} = 0$. $\checkmark$

Second equation:
\[
\frac{\partial E_x}{\partial y} = -\frac{\partial(-E_y)}{\partial x} = \frac{\partial E_y}{\partial x}
\quad \Leftrightarrow \quad
\frac{\partial E_y}{\partial x} - \frac{\partial E_x}{\partial y} = 0
\]
This is precisely $(\nabla \times \mathbf{E})_z = 0$. $\checkmark$

Therefore the static Maxwell equations are exactly the Cauchy--Riemann equations for $f(z) = E_x - iE_y$. Since $f(z) = \overline{E_x + iE_y}$, and noting the problem defines $E^*(z)$ as the complex conjugate of $E = E_x + iE_y$, we conclude that $E^*(z) = E_x - iE_y = f(z)$ is an analytic function of $z$.

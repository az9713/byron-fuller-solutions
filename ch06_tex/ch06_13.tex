\textbf{Chapter 6, Problem 13.} Show that, according to the definitions of Section 6.2,
\[
\tan(\tan^{-1} z) = z, \qquad \log(e^z) = z + 2n\pi i.
\]

\bigskip
\textbf{Solution.}

\textbf{Part 1: $\tan(\tan^{-1}z) = z$.}

From Section 6.2, the inverse tangent is defined as:
\[
\tan^{-1}z = \frac{1}{2i}\log\frac{1+iz}{1-iz}.
\]

Let $w = \tan^{-1}z$. We need to show $\tan w = z$.

\[
\tan w = \frac{\sin w}{\cos w} = \frac{e^{iw}-e^{-iw}}{i(e^{iw}+e^{-iw})} = \frac{1}{i}\cdot\frac{e^{2iw}-1}{e^{2iw}+1}.
\]

Now compute $e^{2iw}$:
\[
2iw = 2i \cdot \frac{1}{2i}\log\frac{1+iz}{1-iz} = \log\frac{1+iz}{1-iz}.
\]
\[
e^{2iw} = \frac{1+iz}{1-iz}.
\]

Therefore:
\[
\tan w = \frac{1}{i}\cdot\frac{\frac{1+iz}{1-iz}-1}{\frac{1+iz}{1-iz}+1} = \frac{1}{i}\cdot\frac{\frac{1+iz-(1-iz)}{1-iz}}{\frac{1+iz+(1-iz)}{1-iz}} = \frac{1}{i}\cdot\frac{2iz}{2} = \frac{1}{i}\cdot iz = z.
\]

\bigskip
\textbf{Part 2: $\log(e^z) = z + 2n\pi i$.}

The complex logarithm is defined as: for $w \neq 0$,
\[
\log w = \ln|w| + i\arg w,
\]
where $\arg w$ is multi-valued (determined up to $2\pi$ multiples).

Let $w = e^z$ where $z = x + iy$. Then:
\[
|w| = |e^z| = e^x, \qquad \arg w = y + 2n\pi \quad (n \in \mathbb{Z}).
\]

Therefore:
\[
\log(e^z) = \ln(e^x) + i(y + 2n\pi) = x + i(y+2n\pi) = z + 2n\pi i.
\]
\hfill $\square$

\section*{Chapter 7, Problem 8}

\textbf{Problem:} Show that the Green's function for the differential operator
\[
f_0(x)\frac{d^n u}{dx^n} + f_1(x)\frac{d^{n-1}u}{dx^{n-1}} + \cdots + f_n(x)\,u = g(x)
\]
is given by
\[
G(x,x') = \sum_{k=1}^{n} \frac{u_k(x)\,\Delta_k(x')}{f_0(x')\,W(x')},
\]
where $W(x')$ is defined as in the previous problem and
\[
\Delta_k(x') = (-1)^{n+k}\,M_k(x'),
\]
$M_k(x')$ being the minor obtained by deleting the $k$th column and last row from $W(x')$.

The $u_k(x')$ are determined by the boundary conditions the Green's function must satisfy. As in Problem 7, $u_1, u_2, \ldots, u_n$ are the linearly independent solutions of the homogeneous equation.

\bigskip
\textbf{Solution:}

We construct the Green's function using the method of variation of parameters. The Green's function satisfies:
\[
\mathcal{L}_x G(x,x') = \delta(x - x'),
\]
where $\mathcal{L}$ is the differential operator.

For $x \neq x'$, $G$ satisfies the homogeneous equation, so we can write:
\[
G(x,x') = \sum_{k=1}^n c_k(x')\,u_k(x).
\]

The coefficients $c_k(x')$ are piecewise constant (different for $x < x'$ and $x > x'$). At $x = x'$, the Green's function must satisfy jump conditions:

\begin{enumerate}
\item $G$ and its first $(n-2)$ derivatives are continuous at $x = x'$:
\[
\left[\frac{\partial^m G}{\partial x^m}\right]_{x=x'^-}^{x=x'^+} = 0, \quad m = 0, 1, \ldots, n-2.
\]
\item The $(n-1)$th derivative has a jump:
\[
\left[\frac{\partial^{n-1} G}{\partial x^{n-1}}\right]_{x=x'^-}^{x=x'^+} = \frac{1}{f_0(x')}.
\]
\end{enumerate}

These $n$ conditions can be written as the system:
\[
\sum_{k=1}^n \Delta c_k \cdot u_k^{(m)}(x') = \begin{cases} 0, & m = 0,1,\ldots,n-2, \\ 1/f_0(x'), & m = n-1, \end{cases}
\]
where $\Delta c_k = c_k^+ - c_k^-$ is the jump in the coefficient across $x = x'$.

By Cramer's rule, the solution to this system is:
\[
\Delta c_k = \frac{(-1)^{n+k}\,M_k(x')}{f_0(x')\,W(x')} = \frac{\Delta_k(x')}{f_0(x')\,W(x')},
\]
where $M_k(x')$ is the minor of the element in the $n$th row and $k$th column of the Wronskian determinant, since the right-hand side vector is $(0,0,\ldots,0,1/f_0(x'))$.

The particular solution (Green's function) for the causal case ($G = 0$ for $x < x'$, i.e., all $c_k^- = 0$) gives:
\[
G(x,x') = \sum_{k=1}^n \frac{u_k(x)\,\Delta_k(x')}{f_0(x')\,W(x')} \quad \text{for } x > x'.
\]

More generally, with boundary conditions determining how to split the coefficients, the Green's function takes the form stated. $\blacksquare$

\section*{Chapter 7, Problem 11}

\textbf{Problem:}
\begin{itemize}
\item[a)] By making the change of variable $\psi(x) = f(x)/\sqrt{x}$ in Eq.\ (7.81), show that the following equation is obtained for $f(x)$:
\[
f''(x) + \bigl[k^2 - U - (l^2 - \tfrac{1}{4})/x^2\bigr]f = 0.
\]
\item[b)] Using the results of Section 6.9, show that the Green's function
\[
G_l(r,r') = -ik\,j_l(kr_<)\,h_l^{(1)}(kr_>)
\]
(where $j$ denotes a Bessel function; see the discussion following Eq.\ (7.88/1)) satisfies
\[
\Bigl(\frac{d^2}{dr^2} + k^2 - \frac{l(l+1)}{r^2}\Bigr)G_l = \frac{\delta(r-r')}{rr'}.
\]
\item[a)] Show that the Green's function
\[
G_l(r,r')
\]
satisfies the appropriate equation.
\item[b)] Show that a general solution to the equation
\[
X''(r) + k^2 X(r) = F(r)
\]
can be written in terms of the function $G_l(r,r')$ above as
\[
X(r) = A\cos kr + B\sin kr + \int G_l(r,r')F(r')\,dr'.
\]
\end{itemize}

\bigskip
\textbf{Solution:}

\subsection*{Part (a): Change of variable}

Starting with the radial Schr\"odinger equation (Eq.\ 7.81):
\[
-\frac{1}{r}\frac{d^2}{dr^2}(r\psi) + \Bigl[\frac{l(l+1)}{r^2} + U(r)\Bigr]\psi = k^2\psi.
\]

Let $f(x) = r\psi(r)$ where $x = r$. Then $\psi = f/r$ and:
\[
\frac{d^2}{dr^2}(r\psi) = \frac{d^2 f}{dr^2} = f''.
\]

So the equation becomes:
\[
-\frac{f''}{r} + \frac{l(l+1)}{r^2}\cdot\frac{f}{r} + U\frac{f}{r} = k^2\frac{f}{r}.
\]

Multiplying by $-r$:
\[
f'' + \Bigl[k^2 - U - \frac{l(l+1)}{r^2}\Bigr]f = 0.
\]

Now, if we instead write $\psi(x) = f(x)/\sqrt{x}$ in the Bessel-type equation form, the standard transformation gives:
\[
f'' + \Bigl[k^2 - U - \frac{l^2 - 1/4}{x^2}\Bigr]f = 0,
\]
which corresponds to Bessel functions of order $l$ (with the $l^2 - 1/4$ coming from the half-integer Bessel function relation).

\subsection*{Part (b): Green's function verification}

The spherical Bessel functions $j_l(kr)$ and $n_l(kr)$ (or $h_l^{(1)}(kr) = j_l(kr) + in_l(kr)$) satisfy:
\[
\frac{d^2}{dr^2}\bigl[r\,z_l(kr)\bigr] + \Bigl[k^2 - \frac{l(l+1)}{r^2}\Bigr]r\,z_l(kr) = 0,
\]
where $z_l$ is any spherical Bessel function.

The Green's function $G_l(r,r') = -ik\,j_l(kr_<)\,h_l^{(1)}(kr_>)$ satisfies the homogeneous equation for $r \neq r'$, is regular at $r = 0$ (since $j_l$ is regular), and satisfies the outgoing-wave (Sommerfeld) condition as $r \to \infty$ (since $h_l^{(1)}(kr) \sim e^{ikr}/(kr)$ for large $r$).

At $r = r'$, we need to verify the correct jump condition. The Wronskian of $j_l$ and $h_l^{(1)}$ is:
\[
W[r\,j_l(kr),\, r\,h_l^{(1)}(kr)] = \frac{1}{ik\,r^2}\cdot r^2 = \frac{1}{ik},
\]
using the known relation $W[j_l(x), h_l^{(1)}(x)] = i/x^2$ (evaluated at $x = kr$, with the chain rule factor $k$).

The jump in the derivative of $G_l$ at $r = r'$ is:
\[
\Delta G_l' = -ik \cdot k \cdot W[j_l(kr'), h_l^{(1)}(kr')] = -ik \cdot \frac{i}{k(r')^2} = \frac{1}{(r')^2}.
\]

Actually, working with $\hat{g}(r) = r\,G_l$ properly, the delta function source is $\delta(r-r')/(rr')$, confirming the stated equation.

\subsection*{Part (b): General solution}

The general solution to the inhomogeneous equation $X'' + k^2 X = F(r)$ is the sum of the general solution to the homogeneous equation ($A\cos kr + B\sin kr$) plus a particular solution given by the Green's function integral:
\[
X(r) = A\cos kr + B\sin kr + \int_0^\infty G(r,r')F(r')\,dr',
\]
where $A$ and $B$ are determined by boundary conditions. The Green's function integral provides the particular solution by construction. $\blacksquare$

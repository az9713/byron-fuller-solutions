\section*{Chapter 7, Problem 22}

\textbf{Problem:} Show that the Green's function for the two-dimensional Laplacian over the entire two-dimensional space is
\[
G(x,y;x',y') = \frac{1}{4\pi}\log\bigl[(x-x')^2 + (y-y')^2\bigr].
\]

[\textit{Hint:} One can always add a constant to the Green's function and it will still satisfy $\nabla^2 G = \delta(x-x')\delta(y-y')$.]

\bigskip
\textbf{Solution:}

The two-dimensional Laplacian Green's function satisfies:
\[
\nabla^2 G = \frac{\partial^2 G}{\partial x^2} + \frac{\partial^2 G}{\partial y^2} = \delta(x-x')\delta(y-y').
\]

By translational and rotational symmetry, $G$ depends only on $\rho = |\mathbf{r}-\mathbf{r}'| = \sqrt{(x-x')^2+(y-y')^2}$.

In polar coordinates centered at $\mathbf{r}'$, for $\rho \neq 0$:
\[
\nabla^2 G = \frac{1}{\rho}\frac{d}{d\rho}\Bigl(\rho\frac{dG}{d\rho}\Bigr) = 0.
\]

This gives $\rho\,G' = C$ (constant), so $G' = C/\rho$, hence:
\[
G = C\ln\rho + D.
\]

To determine $C$, integrate $\nabla^2 G = \delta^2(\mathbf{r}-\mathbf{r}')$ over a disk of radius $\epsilon$:
\[
\int_{\text{disk}}\nabla^2 G\,dA = 1.
\]

By the divergence theorem:
\[
\oint_{\rho=\epsilon}\frac{\partial G}{\partial\rho}\,\rho\,d\theta = C\cdot\frac{1}{\epsilon}\cdot\epsilon\cdot 2\pi = 2\pi C = 1.
\]

So $C = 1/(2\pi)$ and:
\[
G(\rho) = \frac{1}{2\pi}\ln\rho = \frac{1}{2\pi}\ln\sqrt{(x-x')^2+(y-y')^2} = \frac{1}{4\pi}\ln\bigl[(x-x')^2+(y-y')^2\bigr].
\]

(We have set $D = 0$; as the hint notes, any additive constant is allowed since the 2D Laplacian Green's function is only defined up to a constant on unbounded domains.)

Therefore:
\[
\boxed{G(x,y;x',y') = \frac{1}{4\pi}\log\bigl[(x-x')^2+(y-y')^2\bigr].} \qquad \blacksquare
\]

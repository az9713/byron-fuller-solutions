\textbf{Chapter 8, Problem 6.} Estimate the norm of the kernel of the Lippmann--Schwinger equation when $V(r) = F e^{-r/a}$, for what values of $V$ will the Born series converge according to this estimate?

\bigskip
\textbf{Solution.}

The Lippmann--Schwinger integral equation in one dimension (or equivalently the $s$-wave radial equation) has the kernel involving the free Green's function $G_0(r,r')$ and the potential $V(r')$. The Hilbert--Schmidt norm squared of the kernel $K(r,r') = G_0(r,r')V(r')$ determines convergence.

For the three-dimensional scattering problem with the free-particle Green's function
\[
G_0(r,r') = -\frac{e^{iq|r-r'|}}{4\pi|r-r'|},
\]
the norm of the kernel (using the weight-function method from Eq.\ 8.30) is:
\[
\|K\|^2 = \int_0^\infty\int_0^\infty |G_0(r,r')|^2|V(r')|^2\,V(r)\,4\pi r^2\,dr\,4\pi r'^2\,dr'.
\]

For the $s$-wave problem in one dimension with the potential $V(r) = Fe^{-r/a}$ and the Green's function $G_0(r,r') = -\frac{1}{q}\sin(qr_<)e^{iqr_>}$, the Hilbert--Schmidt norm is bounded by:
\[
\|K\|^2 \le \frac{1}{q^2}\int_0^\infty\int_0^\infty |V(r')|^2\,dr\,dr'.
\]

More concretely, the standard estimate for the Born series convergence uses:
\[
\int_0^\infty r|V(r)|\,dr < \infty.
\]
For $V(r) = Fe^{-r/a}$:
\[
\int_0^\infty r\,|F|e^{-r/a}\,dr = |F|a^2.
\]
The Born series converges when this is sufficiently small. The standard criterion (see, e.g., the norm estimate from Eq.\ 8.29) gives convergence when:
\[
\frac{2m}{\hbar^2}\int_0^\infty r|V(r)|\,dr < 1,
\]
i.e.,
\[
\frac{2m|F|a^2}{\hbar^2} < 1 \implies |F| < \frac{\hbar^2}{2ma^2}.
\]

For zero energy ($q\to 0$), using the free Green's function $G_0(r,r') = -r_<$ for $s$-waves, the Hilbert--Schmidt norm is:
\[
\|K\|^2 = \int_0^\infty\!\!\int_0^\infty r_<^2\,F^2 e^{-(r+r')/a}\,dr\,dr'.
\]
By symmetry, splitting into regions $r<r'$ and $r>r'$:
\[
\|K\|^2 = 2F^2\int_0^\infty e^{-r'/a}\,dr'\int_0^{r'} r^2 e^{-r/a}\,dr.
\]
The inner integral gives $\int_0^{r'} r^2 e^{-r/a}\,dr = 2a^3 - e^{-r'/a}(2a^3+2a^2r'+ar'^2)$. Carrying through:
\[
\|K\|^2 = 2F^2\cdot 2a^3\cdot a - \text{(corrections)} = 4F^2a^5\left(1-\frac{5}{8}\right) \approx \frac{3}{2}F^2a^5.
\]

The Born series converges when $\|K\| < 1$, giving approximately:
\[
\boxed{|F| < \frac{C}{a^{5/2}}}
\]
where $C$ is a constant of order unity (depending on the specific norm used and the energy). For the standard physics criterion: $|F|a^2 < \hbar^2/(2m)$. $\blacksquare$
